# rosbag_to_csv

A GUI tool to rosbag topics to csv files

# Install

clone this repository

> $ https://github.com/AtsushiSakai/rosbag_to_csv.git

# How to use

## Start the node

> $ rosrun rosbag_to_csv rosbag_to_csv.py
